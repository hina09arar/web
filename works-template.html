Difference of values

ここでは、日本とドイツの価値観の違いについて説明していきます。

Reference URL

https://www.startup-dojo.com/germany-culture/　https://harubobo.com/2016/04/21/differences-japan-and-germany/

About the difference

⓵「協力」を大事にする日本人、「個性」を大事にするドイツ人 　→　日本人は他人の意見に賛成しがちですが、ドイツでは個人の意見が強調されます。 ⓶新しいものをよく買う日本人、使い勝手のいいものを最後まで使うドイツ人 → ドイツ人は節約することが美徳と考えており、物を大切にするため日本人より消費が少ないです。

Summary

日本とドイツは似ているといわれることが多いですが、実際には多くの違いがあります。
